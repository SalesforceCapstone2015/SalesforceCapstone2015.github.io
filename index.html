Silent Auctions In The 21st Century!

Currently, silent auctions are still held using the traditional pen and paper method. Three teams developed alternative methods for bidding at silent auctions. These methods involve using mobile devices to participate in the auction. By using these apps, non-profit organizations can increase participation to acquire more support for their causes.

The Teams

ItemHunter by Trogdor!!

ItemHunter uses the force.com platform as a backend to host auctions. Bidders use an Android application to interface with the auction.

Installation

bidfresh by Schoolyard Booleans

The bidfresh silent auction platform has an iPhone application for the bidder's interface, and uses a Nodejs server interacting with Salesforce APEX for the administrator web console.
