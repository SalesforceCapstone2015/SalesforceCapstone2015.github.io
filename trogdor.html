ItemHunter by Trogdor!!

ItemHunter consists of two components: the admin console and the bidder app.

Silent auction administrators can use the admin console to set auction and item info, keep track of item statuses, and checkout bidders at the end of an auction.

Bidders use the mobile app to view and bid on items at the auction. The app enables them to keep track of their bids and be notified when another participant outbids them on an item.

Bidder app installation (Android only)

To install the app...

  1. Request beta access from Joel (jnwilcox@calpoly.edu) or Sterling (starng@calpoly.edu).
  2. You will receive an invite to the Trogdor SE Capstone Google group. Accept the invite.
  3. Use your internet browser of choice to access the Google group at https://groups.google.com/forum/#!forum/trogdor-se-capstone.
  4. Click on the Play Store link within the group.
  5. You are now ready to install the app via the Play Store.

Salesforce Backend / Admin Console Installation

Go to the app exchange

  1. Create a new org
  2. Install the alpha package
  3. Create the ItemHunter site
    1. Go to setup -> develop -> sites
    2. Enter a subdomain.
    3. click "new" to create a new site
    4. Enter "ItemHunter" as the site label, site name, and default web address
    5. specify the "Active Site Home Page" as any page… the value doesn't matter
    6. make sure the site is "active" and that "require https" is enabled
    7. on your new site, goto Public Access Settings -> enabled apex class access (near bottom)
    8. add every class that begins with "Endpoint" (except the Test classes)
  4. Add your site's subdomain as a custom setting value
    1. Go to setup -> develop -> custom settings
    2. click "ItemHunterSite"
    3. click "manage"
    4. click "new"
    5. as "name" enter "subdomain"
    6. as "value" enter "your-subdomain-.na#", eg "josh-test-ih-developer-edition.na24"
  5. Select "Item Hunter" as the app (top right app-selection drop-down)
  6. Click on "Admin Console" in the page tabs
  7. Enjoy!
